    Section 10.2 Outer measure

    Remember that our goal is to find a more general way to measure sets of real numbers. In this section, we’ll see how we can take the idea of measuring intervals and use it to begin measuring some other sets.
    🔗
    First, here’s a variation on the idea of an open cover, which we first saw in Section 5.3:
    🔗

    Definition 10.2.1. Countable open cover.

    A countable open cover of a set \(A\) is a countable collection of open intervals \(\{I_{n}\}\) where
    \begin{equation*} A \subseteq \displaystyle\bigcup_{n} I_{n}. \end{equation*}
    🔗
    🔗
    The only difference between a “countable open cover” and a plain-old “open cover” is that a countable open cover can only have countably many sets in it.

    Aside: A detail about open sets.

    Often the definition of “open cover” says that the \(I_{n}\)’s must be open sets. But by Theorem 5.5, every open set is a countable union of open intervals. That, plus the fact that the union of countable sets is countable, lets us assume that each \(I_{n}\) in a countable open cover is just one open interval. So with a countable open cover, we’re looking at sets of open intervals that completely cover a set \(A\text{,}\) as long as there aren’t too many such sets.
    🔗
    A regular open cover could have uncountably many sets.
    🔗

    Example 10.2.2. Four of open covers of \([0,1]\).

    Here are four examples of open covers of \([0,1]\text{:}\)
    🔗
    1. \(\displaystyle C = \{ (x - 0.1, x + 0.1 ) : x \in [0,1] \}\)
      🔗
    2. \(\displaystyle D = \{ (x - 0.1, x + 0.1) : x \in [0,1] \cap \Q \}\)
      🔗
    3. \(\displaystyle E = \{ (-0.1, 1.1) \}\)
      🔗
    4. \(\displaystyle F = \{ ( \frac{1}{n}, 1 - \frac{1}{n} ) : n \in \N\}\)
      🔗
    All of these are open covers of \([0,1]\text{.}\) However, only \(D\text{,}\) \(E\text{,}\) and \(F\) are countable open covers, since \(|D| = |\Q| = |\N|\text{,}\) \(|E| = 1\text{,}\) and \(|F| = |\N|\text{.}\) On the other hand, \(|C| = |\R|\) so \(C\) is uncountably large and therefore not a countable open cover.
    🔗
    🔗
    We use this idea of being “covered by open intervals” to define our first attempt at a measure:
    🔗

    Definition 10.2.3. Outer measure.

    The outer measure of a set \(A\text{,}\) denoted \(\meso{A}\text{,}\) is defined as:
    \begin{equation*} \meso{A} = \inf_{A \subseteq \cup \{I_{n}\}} \sum \len{I_{n}}, \end{equation*}
    where the infimum runs over all countable open covers \(\{I_n\}\) of \(A\text{.}\) It is possible that the infimum is infinitely large, that is, that \(\meso{A} = \infty\text{.}\)
    🔗
    🔗
    There’s a lot packed in to this definition.

    Aside: We’re not telling you the whole story.

    The function \(\meson\) is more properly called the Lebesgue outer measure. There are other outer measures, but this is the only one we’ll study. Plus Lebesgue will get plenty of name-checks later on.
    🔗
    The idea is that, if we can cover a set with open intervals, then the sum of the lengths of those intervals is an overestimate for the set’s true measure (that’s the “outer” part). Note that the intervals need not be disjoint. But there are lots of ways to cover a set with open intervals, and some of them waste a lot of space while others waste very little. If we take the infimum across all possible ways to cover the set with open intervals, then we hopefully get close to the true measure of the set. Let’s look at some examples.
    🔗

    Example 10.2.4. Outer measure agrees with measure zero.

    The definition of “outer measure” agrees with our earlier definition of “measure zero” (Definition 8.23) in the case \(\meso{A} = 0\text{.}\)
    🔗
    To see this, first suppose a set \(A\) has measure zero. Then for any \(\ve > 0\) there exists a countable open cover \(\{ J_n \}\) of \(A\) with \(\sum \ell(J_{n}) \lt \ve\text{.}\) Thus
    \begin{equation*} \displaystyle\inf_{A \subseteq \cup \{I_{n}\}} \sum \ell(I_{n}) \leq 0\text{,} \end{equation*}
    because the infimum across all countable open covers includes this specific open cover \(\{ J_n \}\text{.}\) Because the lengths of intervals are always nonnegative, this infimum actually equals zero, which means that \(\meso{A} = 0\text{.}\)
    🔗
    Conversely, if \(\meso{A} = 0\) then by the Suprema Analytically theorem (1.24), for any \(\ve > 0\) there exists a countable open cover such that \(\sum \ell(I_{n}) \lt 0 + \ve\text{,}\) which is exactly what the definition of “measure zero” requires.
    🔗
    🔗

    Example 10.2.5. The outer measure of \([0,1]\).

    Let \(A = [0,1]\text{.}\) We really really want \(\meso{A} = 1\) or else this outer measure idea won’t make any sense at all. So let’s show that it works as expected. Buckle in.
    🔗
    First, let \(\ve > 0\) and consider the interval \((0-\ve, 1 + \ve)\text{.}\) Then \(\{ (0-\ve, 1 + \ve) \}\) is a countable open cover of \([0,1]\text{.}\) We also know that \(\len{0-\ve, 1 + \ve} = 1+\ve - (0-\ve) = 1 + 2\ve\text{.}\) This means that \(\meso{A} \leq 1 + 2\ve\) for all \(\ve > 0\text{,}\) or in other words, \(\meso{A} \leq 1\text{.}\) So far, so good.
    🔗
    Next if we can show that \(\meso{A} \geq 1\text{,}\) then we’ll have \(\meso{A} = 1\text{.}\) Start by considering any countable open cover of \([0,1]\text{.}\) Because \([0,1]\) is closed and bounded, by the Heine-Borel Theorem, it is compact (You didn’t see that one coming, did you?). By definition of “compact,” every open cover of \([0,1]\) has a finite subcover \(\mathcal{I} = \{I_{1}, I_{2}, \ldots, I_{n}\}\text{.}\) That is, \(\mathcal{I}\) is a finite list of open intervals that cover \([0,1]\text{.}\)
    🔗
    Next, we’ll turn this finite cover of \([0,1]\) into a minimal cover, by selecting only a bare minimum of intervals from \(\mathcal{I}\text{.}\) Because \(\mathcal{I}\) covers \([0,1]\text{,}\) there must be some interval \((a_{1}, b_{1}) \in \mathcal{I}\) that covers 0, that is, \(0 \in (a_{1}, b_{1})\text{.}\) Perhaps \((a_{1},b_{1})\) covers all of \([0,1]\text{,}\) in which case we stop here. But if it doesn’t, that means \(b_{1} \in (0,1]\text{.}\) Then we can find another interval \((a_{2},b_{2}) \in \mathcal{I}\) so that \(b_{1} \in (a_{2}, b_{2})\text{.}\) Again, perhaps \(b_{2} > 1\) in which case we’ve covered all of \([0,1]\text{.}\) If so, we stop. If not, we continue this process, finding a sequence of intervals from \(\mathcal{I}\text{:}\)
    🔗
    \begin{equation*} (a_{1}, b_{1}), (a_{2}, b_{2}), \ldots, (a_{k},b_{k}) \end{equation*}
    so that each \(b_{i-1} \in (a_{i}, b_{i})\) and together the intervals completely cover \([0,1]\text{,}\) that is, \([0,1] \subseteq \cup_{i=1}^{k} (a_{i}, b_{i})\text{.}\) Notice that we can do this because \(\mathcal{I}\) covers \([0,1]\text{,}\) and this process must end with a finite number of intervals because \(\mathcal{I}\) is finite by assumption.
    🔗
    Now let’s calculate the total length of these intervals:
    🔗
    \begin{align} \sum_{i=1}^{k} \len{a_{i},b_{i}} \amp= (b_{1} - a_{1}) + (b_{2} - a_{2}) + \cdots + (b_{k-1} - a_{k-1}) + (b_{k}-a_{k})\notag\\ \amp= (b_{k} - a_{k}) + (b_{k-1} - a_{k-1}) + \cdots + (b_{2} - a_{2}) + (b_{1}-a_{1})\notag\\ \amp= b_{k} - (a_{k} - b_{k-1}) - (a_{k-1} - b_{k-2}) - \cdots - (a_{i} - b_{i-1}) - \cdots - (a_{2} - b_{1}) - a_{1}.\tag{10.2.1} \end{align}
    The second line just writes the first line in reverse order. The third line is a clever trick: We regroup the \(a\)’s and \(b\)’s, shifting each set of parentheses one term to the right and attending really carefully to negative signs.
    🔗
    Remember that we chose these intervals to overlap, which we wrote as: \(b_{i-1} \in (a_{i}, b_{i})\text{.}\) Thus \(a_{i} \lt b_{i-1}\text{,}\) and so each regrouped term in (10.2.1) is negative: \(a_{i} - b_{i-1} \lt 0\text{.}\) So each term subtracts a negative amount, also known as adding a positive amount: \(-(a_{i}-b_{i-1}) > 0\text{.}\) That lets us make a big estimate:
    🔗
    \begin{equation*} b_{k} - (a_{k} - b_{k-1}) - (a_{k-1} - b_{k-2}) - \cdots - (a_{i} - b_{i-1}) - \cdots - (a_{2} - b_{1}) - a_{1} > b_{k} - a_{1}. \end{equation*}
    But we also know that \(b_{k} > 1\) and \(a_{1} \lt 0\text{.}\) Putting this all together,
    🔗
    \begin{equation*} \sum_{i=1}^{k} \len{a_{i},b_{i}} > b_{k} - a_{1} > 1 - 0 = 1. \end{equation*}
    So the total length of this particular subcover \(\mathcal{I}\) is greater than 1. That means that the total length of the original cover is also greater than 1. Finally, remember that \(\meson\) is an infimum calculated across all possible countable open covers \(\mathcal{I}\text{.}\) Since each set in that infimum leads us to a total length greater than 1, the infimum itself must be greater than or equal to 1. (We proved this somewhere back in Chapters 1--7, I’m sure of it.) In other words, \(\meso{[0,1]} \geq 1\text{.}\)
    🔗
    Putting together the two inequalities, we must have that \(\meso{[0,1]} = 1\text{.}\)
    🔗
    🔗
