    Section 10.1 Measuring Sets

    Here’s a reminder of Definition 8.22 with slightly different notation:
    🔗

    Definition 10.1.1. Interval length.

    The length of the interval \([a,b]\) is equal to \(b-a\text{,}\) and is denoted \(\len{[a,b]}\text{.}\) This is also the length of \((a,b)\text{,}\) \((a,b]\text{,}\) and \([a,b)\text{.}\) Intervals that involve \(\pm \infty\) are defined to have length \(\infty\text{.}\)
    🔗
    🔗
    This is a nice and simple way to measure the length of one interval. What about sets that aren’t intervals? In this chapter, we’ll see how to do exactly that.
    🔗
    For example, suppose \(I = (-1,1) \cup [2,5) \cup [7,8]\text{.}\) We would like the length of \(I\) to be 6, since that’s the sum of the lengths of the intervals that make it up. Note that \(\ell\text{,}\) as we’ve defined it, doesn’t allow us to do this. That isn’t too hard to fix, but it still needs to be fixed. On the other hand, what should we call the length of \(\Q\text{?}\) This doesn’t contain any intervals but is also infinitely large. As we’ve seen before, the length (really size) of \(\Q\) is in fact zero, at least under one possible definition.
    🔗
    The new idea we need is called a measure. A measure is a set function, that is, a function whose domain includes subsets of \(\R\text{.}\) Its codomain is the extended real numbers, that is, \(\R \cup \{\pm \infty\}\text{.}\) The extended real numbers let us say, as we want to, that the measure of some sets is \(\infty\text{.}\)
    🔗
    Here are some nice properties that we would like any measure \(\mesn\) to have:
    🔗
    1. \(\mes{A}\) is defined for every subset \(A \subseteq \R\text{.}\) In other words, the domain of \(\mesn\) is \(\mathcal{P}(\R)\text{,}\) the power set of the real numbers.
      🔗
    2. For any interval \(I\text{,}\) \(\mes{I} = \len{I}\text{,}\) and \(\mes{\emptyset} = 0\) too.
      🔗
    3. \(\mesn\) is countably additive. That is, if \(\{A_{n}\}\) is a countable collection of disjoint sets, then \(\mes{\cup A_{n}} = \sum \mes{A_{n}}\text{.}\)
      🔗
    4. \(\mesn\) is invariant under translation. That is, if \(A\) is some set, \(x \in \R\text{,}\) and \(A + x = \{a + x : a \in A\}\text{,}\) then \(\mes{A} = \mes{A+x}\text{.}\)
      🔗
    Go back and think about what each of these properties means and why it is important. Once you’ve done that, here’s an explanation (don’t continue until you’ve thought through each property!):
    🔗
    1. We want to be able to measure every possible set.
      🔗
    2. A measure should give the result we expect for intervals: the interval’s length.
      🔗
    3. If a set is made up of a bunch of disjoint parts, then we can measure those parts separately and add them up to get the set’s entire measure.
      🔗
    4. This one requires some more explanation. Think about the interval \(A = [0,1]\text{.}\) Then \(A + 3 = \{a + 3 : a \in A\} = [3,4]\text{.}\) You can imagine this as sliding (aka translating) \(A\) by 3 units to the right on a number line. This item says that translating \(A\) shouldn’t change the measure of \(A\text{,}\) because we haven’t changed anything important about how the numbers in \(A\) are arranged.
      🔗
    Unfortunately, this is a fool’s errand: no set function can satisfy all four of these properties.

    Aside: Wait, what?

    Amazingly, it’s not even known if there is a set function that satisfies just properties #1, #2, and #3 at the same time.
    🔗
    The best solution is to give ourselves some wiggle room on #1: We allow a measure to be defined only on certain subsets of the real numbers, but not all of them. This unfortunately means that there will always be some nonmeasurable sets. We would nonetheless like to be able to measure as many sets as possible.
    🔗
    We do require a little bit of structure, however. Here’s an important definition:
    🔗

    Definition 10.1.2. \(\sigma\)-algebra.

    A \(\sigma\)-algebra (pronounced “sigma-algebra”) is a collection \(\mathcal{M}\) of subsets of \(\R\) that satisfies the following properties:
    🔗
    1. Closure under complements: If \(A \in \mathcal{M}\) then \({A^{c}} \in \mathcal{M}\text{.}\)
      🔗
    2. Closure under countable unions: If \(\{A_{n}\}\) is a countable collection of subsets in \(\mathcal{M}\text{,}\) then \(\cup A_{n} \in \mathcal{M}\text{.}\)
      🔗
    3. Closure under countable intersections: If \(\{A_{n}\}\) is a countable collection of subsets in \(\mathcal{M}\text{,}\) then \(\cap A_{n} \in \mathcal{M}\text{.}\)
      🔗
    🔗
    In the above definitions, “countable” also includes “finite”. For example, property #2 implies that if \(A\) and \(B\) are subsets in \(\mathcal{M}\text{,}\) then \(A \cup B \in \mathcal{M}\) too.
    🔗
    This definition is pretty abstract, but the good news is that you’re already familiar with at least one \(\sigma\)-algebra:
    🔗

    Example 10.1.3. \(\mathcal{P}(\R)\) is a \(\sigma\)-algebra.

    The set of all subsets of \(\R\text{,}\) that is \(\mathcal{P}(\R)\text{,}\) is a \(\sigma\)-algebra. For any given collection of sets of real numbers, their unions, intersections, and complements are also a set of real numbers (possibly an empty one).
    🔗
    🔗
    Here’s a slightly more interesting example:
    🔗

    Example 10.1.4. A finite \(\sigma\)-algebra.

    Let \(\mathcal{M} = \{\emptyset, [0,1], (-\infty,0) \cup (1,\infty), \R\}\text{.}\) Then \(\mathcal{M}\) is a \(\sigma\)-algebra. You should double-check each property.
    🔗
    🔗
    The way to think about a \(\sigma\)-algebra is that it’s a bunch of sets in which you can take complements, unions, and intersections without having to worry about somehow ending up with a set that’s not allowed. There is one limit: You can’t take too “big” of an intersection or union -- only countably many sets at a time.
    🔗
    A \(\sigma\)-algebra is exactly what we need to make measure functions work. Unfortunately, it’s too much to ask that we can measure all of \(\mathcal{P}(\R)\text{,}\) but a slightly smaller -- but still ginormous -- collection of sets will be measurable. That set will be a \(\sigma\)-algebra as well. The rest of this chapter is all about how we can do this. We’re finally ready to carefully define a measure:
    🔗

    Definition 10.1.5. Measure function.

    Let \(\mathcal{M}\) be a \(\sigma\)-algebra. A function \(\mesn : \mathcal{M} \to \R \cup \{\pm \infty\}\) is called a measure if it has all of these properties:
    🔗
    1. Non-negativity: \(\mes{A} \geq 0\) for all \(A \in \mathcal{M}\text{.}\)
      🔗
    2. \(\mes{\emptyset} = 0\text{.}\)
      🔗
    3. Countable additivity: For any countable collection \(\{A_{n}\}\) of disjoint sets in \(\mathcal{M}\text{,}\) \(\mes{\cup A_{n}} = \sum \mes{A_{n}}\text{.}\)
      🔗
    🔗
    Some of our “nice properties” are missing from this definition. The definition of measure is very general and applies in situations where intervals don’t even make sense. In the following sections, we will develop an example of a measure that also satisfies \(\mes{I} = \ell(I)\) for each interval and is invariant under translation.
    🔗
